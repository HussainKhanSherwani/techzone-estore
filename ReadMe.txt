To setup this Project you have to do some steps as follows:

Firstly make sure nodejs is installed in your pc then follow these steps:

Go to server folder and follow these steps:
1.go to terminal make sure you are in server folder
2.When you are in server folder run command "npm install express mongoose bcrypt" also "npm i dotenv"
3.Now after all the installations are done in server folder run command "npx nodemon ./index.js"
4.Now server must get connected 

Now Go to client folder and follow these steps:
1.make sure you are in client folder inside terminal
2.Now run command "npm i"
3.After installations run command "npm start"
4 after some time your application is running on localhost:3000
